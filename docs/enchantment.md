# Enchantment

### Fields

   Field   | Type | Default | Description
-----------|------|---------|-------------
`identifier` | [Identifier]() | *mandatory* | The identifier is what the enchant appears as in the /enchant command.

### Example Code

```json
{
	"type": "ccpacks:enchantment",
	"identifier": "example_pack:winged_boost"
}
```

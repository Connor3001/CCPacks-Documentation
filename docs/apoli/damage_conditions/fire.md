# Fire

[Damage Condition](../damage_conditions.md).

Checks whether the damage is fire damage (this includes lava damage!).

Type ID: `origins:fire`

### Fields:

_None._

### Example
```json
"damage_condition": {
    "type": "origins:fire"
}
```
This example checks if the damage taken/dealt is considered fire damage.

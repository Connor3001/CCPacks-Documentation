# Tamed

[Entity Condition](../entity_conditions.md).

Checks whether a tamable entity is tamed. Will return false if the entity is not tamable.

Type ID: `origins:tamed`

### Fields:

_None._

### Example:
```json
"condition": {
    "type": "origins:tamed"
}
```

The sturcture for all tools is the same but it depends on in which folder it is to choose the tool.

```json
{
	"type": "ccpacks:axe",
	"identifier": "example_pack:example_axe",
	"durability": 47,
	"mining_speed_multiplier": 1.5,
	"attack_damage": 13,
	"attack_speed": 1.6,
	"mining_level": 2,
	"enchantability": 7
}
```
